Інструкція розробника

Вступ

Система була розроблена в середовищі Visual Studio Code з використанням мови програмування Python, що забезпечує високий рівень портативності, простоту підтримки та розширення функціональності. Завдяки використанню Python як основної мови, технічні вимоги до комп’ютера, на якому буде встановлюватися та використовуватися утиліта, залишаються мінімальними, що дозволяє запускати її навіть на малопотужних пристроях.

Системні вимоги

  • Операційна система: Windows 7 / 10 / 11 або будь-який дистрибутив Linux (Ubuntu, Debian, Arch, тощо)
  • Оперативна пам’ять: від 4 Гб
  • Вільне місце на диску: від 200 Мб
  • Python: версія 3.9 або вище

Мінімальне програмне забезпечення

  • Python – мова програмування, яка забезпечує виконання логіки редактора
  • Visual Studio Code – середовище розробки, рекомендоване для підтримки, налагодження та модернізації утиліти
  • Модулі Python (встановлюються через pip):
    • PyQt5 – для GUI-інтерфейсу
    • json, sys, pathlib – для базової роботи CLI-інтерфейсу
    • (опційно) pytest – для модульного тестування

Платформна сумісність

CLI-утиліта є кросплатформною, тобто не прив'язана до конкретної операційної системи. Вона однаково ефективно функціонує на Windows та Linux завдяки особливостям Python, який є інтерпретованою мовою. Усі функції файлів, навігації, обробки текстових форматів (JSON) реалізовані з урахуванням незалежності від файлової системи.

Інструкція зі встановлення

  1. Встановити інтерпретатор Python (з сайту python.org).
  2. Клонувати, завантажити проєкт з репозиторію або завантажте цей проект
  3. Встановити необхідні залежності: 
    pip install -r requirements.txt
  4. Запустити утиліту: 
    python cli_editor.py

Модернізація та розширення

Для внесення змін або вдосконалення функціональності достатньо використовувати те ж саме середовище — Visual Studio Code — разом з розширеннями для Python (наприклад, Pylance, Python Debugger). Проєкт має модульну структуру, що дає змогу швидко змінювати або додавати нові функції

  • editor_logic.py — бізнес-логіка, робота з рівнями, об’єктами, файлами
  • gui_editor.py — графічний інтерфейс
  • cli_editor.py — обробка введення користувача через консоль
Завдяки відкритому коду та прозорій архітектурі, утиліта легко адаптується під нові формати (наприклад, YAML або XML), розширюється новими командами, або інтегрується з іншими ігровими компонентами.
