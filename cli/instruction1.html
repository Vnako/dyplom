Інструкція розробника для редактора рівнів гри Slime Quest: Dungeon

Вступ

Програма редактору рівнів розроблена в середовищі Visual Studio Code з використанням мови програмування Python, що забезпечує високий рівень портативності, простоту підтримки та розширення функціональності. Завдяки використанню Python як основної мови, технічні вимоги до комп’ютера, на якому буде встановлюватися та використовуватися утиліта, залишаються мінімальними, що дозволяє запускати її навіть на малопотужних пристроях.

Системні вимоги

  • Операційна система: Windows 7 / 10 / 11 або будь-який дистрибутив Linux (Ubuntu, Debian, Arch, тощо);
  • Оперативна пам’ять: від 4 Гб;
  • Вільне місце на диску: від 200 Мб.

Платформна сумісність

CLI-утиліта є кросплатформною, тобто не прив'язана до конкретної операційної системи. Вона однаково ефективно функціонує на Windows та Linux завдяки особливостям Python, який є інтерпретованою мовою. Усі функції файлів, навігації, обробки текстових форматів (JSON) реалізовані з урахуванням незалежності від файлової системи.
